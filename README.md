📊 TelecomX — Previsão de Churn

Projeto de Machine Learning para prever a evasão de clientes da empresa fictícia TelecomX e identificar fatores relacionados ao cancelamento.

Foram treinados dois modelos de classificação: Logistic Regression e Random Forest. A Logistic Regression apresentou melhor desempenho, alcançando cerca de 80% de acurácia.

A análise mostrou que o churn está mais associado a contratos mensais, menor tempo de relacionamento, pagamento via electronic check e alguns perfis específicos de clientes. Já fatores como contratos de longo prazo, maior tempo de permanência e serviços adicionais estão ligados a maior retenção.

O projeto demonstra como Machine Learning pode ajudar empresas a prever cancelamentos e apoiar estratégias de retenção de clientes.
